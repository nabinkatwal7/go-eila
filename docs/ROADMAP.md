# MyTrack Roadmap

This document outlines the planned future features and improvements for MyTrack.

## Short Term

- [x] **Transaction Management**
    - Edit existing transactions
    - Delete transactions (with confirmation)
    - Search/Filter transactions by date, amount, or description
- [x] **Data Visualization**
    - Implement real charts in Dashboard (replacing placeholders)
    - Net worth progression graph
    - Category breakdown pie chart
- [x] **Import/Export**
    - CSV Import for bank statements
    - Full database backup/restore to a single file

## Medium Term

- [ ] **Recurring Transactions**
    - Automatic creation of monthly bills/subscriptions
    - Reminders for upcoming payments
- [ ] **Advanced Budgeting**
    - Rollover budgets (unused amount moves to next month)
    - Annual budgets
- [ ] **Multi-Currency Support**
    - Live exchange rate updates
    - Multi-currency reporting

## Long Term

- [ ] **Mobile Companion App**
    - Android/iOS app for on-the-go entry
    - Sync via local network or encrypted cloud
- [ ] **Investment Tracking**
    - Stock price updates
    - Portfolio performance analysis
- [ ] **Receipt Scanning**
    - OCR to auto-extract transaction details from receipt images
